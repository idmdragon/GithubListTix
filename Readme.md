
# Project Title

GithubList

## Getting Started

For running the project please replace "API_KEY" in package com.idmdragon.githublist.utils 

### Techstack

- Kotlin
- Dagger2
- Flow
- Retrofit
- Okhttp
- Mvvm with Clean Architecture
- Unit testing with mockito
